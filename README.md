# Restaurant FAQ Bot

## Description
Restaurant FAQ Bot is a simple web-based chatbot that answers common customer questions related to restaurant services such as opening hours, menu availability, location, and delivery options.

## Features
- Answers frequently asked questions
- Simple and easy-to-use interface
- Keyword-based question matching

## Technology Used
- Python
- Flask
- HTML, CSS, JavaScript

## How to Run
1. Install required libraries
   pip install flask
2. Run the application
   python app.py
3. Open your browser and go to
   http://127.0.0.1:5000/
4. open google tab and go to
   https://resturant-faq-bot.onrender.com/
